## 起動方法
コンテナ起動  
`docker-compose -f "docker-compose.yml" up -d --build`  
  
pythonコンテナに接続し  
`cd /home/app/flaskr`  
`python app.py`  
  
ブラウザからアクセス  
`http://localhost:777/`
